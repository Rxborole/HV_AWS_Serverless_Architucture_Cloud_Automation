# Automated S3 Bucket Cleanup Using AWS Lambda

## Objective

Developed a serverless solution using AWS Lambda and Boto3 to automatically remove files older than 30 days from an Amazon S3 bucket.

## Technologies Used

* AWS Lambda
* Amazon S3
* IAM
* CloudWatch
* Python
* Boto3

## Features

* Lists objects in an S3 bucket
* Identifies files older than 30 days
* Deletes outdated files automatically
* Logs deleted objects to CloudWatch
* Supports scheduled execution using EventBridge

## Outcome

Successfully reduced storage clutter and demonstrated serverless automation using AWS services.
